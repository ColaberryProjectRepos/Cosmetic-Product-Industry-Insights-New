Product Count Bar Chart 16

  1. Start by selecting the "Line and Stacked Column Chart" visualization and place it in the top left box of our report, or you can copy paste one of the KPI Cards and Customize it just like in the GIF.
  2. Next, choose the values and measures to be used in this chart. We'll use the "Product Count" measure from the "Main Measures" table, the "Date (Year & Month)" value from the "Calendar" table, and all of the YoY % measures from the "YoY Measures" table.
  3. Remember, just selecting these measures and values isn't enough. Place "Year" on the x-axis, "Product Count" on the y-axis, and all the YoY % values into the line values. Refer to the final screenshot for the correct renaming.
  4. Use conditional formatting for the Title and select the "Dynamic Chart Selected Value" measure from the "Dynamic Measures" table.
  5. For formatting, refer to the GIF provided. It will guide you on how to format the chart effectively. Remember to use the main blue color for "Product" and the same color from the sub KPIs for the lines.
