Creating Date Refresh 12

Alright, let's add a refresh date card. To do this, follow these steps: 
  1. From the Home tab, click "Get Data" and select "Blank Query." Set the value or source to be the function `DateTime.LocalNow()` and rename the query to "Refresh Date." Then, rename the column to "Date time." 
  2. Next, to display it, add a card from the visualizations pane. Follow the GIFs below for implementation: 
    • First, add the date source; then, add the card to the dashboard and set "Date time" as the value. 
    • Finally, customize it using the properties provided in the GIF for the card.
