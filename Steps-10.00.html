Creating Calendar Table 10
We will now proceed to generate a calendar table by extracting it from our data source 'cscpopendata' table utilizing a DAX formula. This process is straightforward. Employ the provided DAX formula to accomplish this task, and refer to the accompanying GIF for step-by-step guidance.

Dax Formula:
Calendar = ADDCOLUMNS (
CALENDAR (Date(Year(Min('cscpopendata'[InitialDateReported])),
Month(Min('cscpopendata'[InitialDateReported])),
DAY(Min('cscpopendata'[InitialDateReported]))),
Date(Year(Max('cscpopendata'[InitialDateReported])),
Month(Max('cscpopendata'[InitialDateReported])),
DAY(Max('cscpopendata'[InitialDateReported])))),
"Month number", FORMAT ([Date], "MM"),
"Month Year", FORMAT ([Date], "MMMYYYY"),
"Month", FORMAT ([Date], "mmmm"),
"Week Number", "Week-"& WEEKNUM([Date]),
"Week", WEEKNUM ([Date]),
"Day Of Month", Month([DATE]) &"-"& FORMAT ( [Date], "dd" ),
"Year Quarter", FORMAT ([Date], "YYYY") & "/Q" & FORMAT ([Date], "Q"),
"Year", FORMAT ([Date], "YYYY")
)
