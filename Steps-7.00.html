Download the Project Essentials ZIP by clicking the Red Icon above!

  1.  Change the default font of the project to "Corbel." This can be done by going to View > Customize Current Theme > Text > set all to "Corbel."
  2.  Follow the GIF file attached to customize the background for the report. This gives a better custom-looking feel to the report while also saving space by not using shapes. (You can also add the globe and Colaberry Logo included in the "Project Essentials" ZIP).
  3.  Import the data by clicking "Import data from Excel" and selecting the "cscpopendata.csv" file included in the "Project Essentials" ZIP. Select "cscpopendata" and click "Transform Data" then move on to Step 3 for more instruction.
